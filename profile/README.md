## 🔥 **𝗣𝗿𝗼𝗷𝗲𝗰𝘁 𝗢𝘃𝗲𝗿𝘃𝗶𝗲𝘄**

🌿 **Plogging(플로깅)**: 조깅하며 쓰레기를 줍는 친환경 활동  

### 𝗕𝗮𝗰𝗸𝗴𝗿𝗼𝘂𝗻𝗱
- **문제점**: 쓰레기 무단 투기로 인한 환경 오염과 플로깅 활동을 체계적으로 지원하는 플랫폼의 부재
- **필요성**: MZ세대 러닝 크루 트렌드에 맞춰 환경을 개선하는 플로깅 활동의 지원이 필요함

### 𝗚𝗼𝗮𝗹𝘀
- 환경 보호: 길가 쓰레기 문제 해결
- 플로깅 활동 지원 및 활성화

### 𝗜𝗻𝘁𝗿𝗼𝗱𝘂𝗰𝘁𝗶𝗼𝗻

- **Plog(플로그)**: 플로깅을 통해 지역 환경 개선을 지원하는 앱
- 사용자는 **쓰레기 다발 위치**를 확인하고  
- **최적의 플로깅 루트**를 추천받으며  
- 활동에 따라 **리워드**를 받을 수 있습니다 🌿
